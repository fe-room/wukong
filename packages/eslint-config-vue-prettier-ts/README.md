# 代码规范
## 依赖库
+ Eslint
+ Prittier
+ Typescript

## 安装

+ `npm i @wkp/eslint-config-vue-prettier-ts --dev` or
+ `yarn add @wkp/eslint-config-vue-prettier-ts -D`


## 配置
+ 在 `.eslintrc.js` 中加入 :
```javascript 
module.exports = {
  "extends": ["@wkp/eslint-config-vue-prettier-ts"]
}
```
