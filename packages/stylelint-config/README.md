# Stylelint Config of wkp

## Install

#### NPM

```shell
npm i @wkp/stylelint-config -D
```

#### YARN

```shell
yarn add @wkp/stylelint-config --dev
```

## Usage

```js
{
  "extends": ["@wkp/stylelint-config"]
}
```
